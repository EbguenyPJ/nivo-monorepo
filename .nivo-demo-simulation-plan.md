- [ ] Curvas de venta: picos Sáb/Dom, valles Martes, quincenas (15 y 30)
- [ ] Catálogos complejos: Listas de precios mayoreo, colores, tallas, impuestos, colecciones
- [ ] Cortes de caja con faltantes/sobrantes
- [ ] Gastos con foto adjunta desde App B2B
- [ ] Rentabilidad variable (70% margen vs 10% liquidación)
- [ ] Reabastecimiento: stock mínimo → Requisición → PO → Recepción
- [ ] CRM + Lealtad: puntos, QR, pagos mixtos
- [ ] Apartados que se vencen
- [ ] Devoluciones y cancelaciones
- [ ] Traspasos entre sucursales
- [ ] Auditorías de inventario (full + partial)
- [ ] Órdenes omnicanal: BOPIS, delivery, ship_to_home

- [ ] Suscripciones: upgrade, downgrade, churn, past_due
- [ ] Pre-ventas con QR

- [ ] Cuentas por pagar a proveedores
- [ ] Facturas de billing
